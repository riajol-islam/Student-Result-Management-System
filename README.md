-----------Student Result Management System-------------------
A complete SQL-based database management system.

🚀 Features
- **50 Student Records:** Comprehensive profiles of CSE students.
- **Academic Tracking:** GPA monitoring across semesters.
- **Course Management:** Structured student-course enrollment.
- **Relational Integrity:** Implemented using Foreign Keys and Cascading.

 🛠️ Tech Stack
- **Database:** MySQL
- **Tool:** MySQL Workbench
